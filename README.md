## 🚀 Deployment

Deployed using **GitHub Pages**:
1. Push to a repository named `elvis-portfolio`
2. Go to Settings → Pages → Source → `main / root`
3. Portfolio will be available at `https://yourusername.github.io/elvis-portfolio/`
